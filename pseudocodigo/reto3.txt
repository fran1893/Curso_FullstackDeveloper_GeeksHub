RETO 3: Crear el pseudocódigo de un programa que almacenen 3 números en 3 variables
A, B y C. El diagrama debe decidir cuál es el mayor y cuál es el menor.


Programa: CualMayor
Entorno: A, B y C son números

    escribir: "Introduzca un segundo numero"
    leer numero y almacenarlo como B

    escribir: "Introduzca un tercer numero"

   IF A > B AND A > C THEN
    escribir: A" es el mayor"
    IF B < C THEN

        escribir: C" es el menor"
    END IF
